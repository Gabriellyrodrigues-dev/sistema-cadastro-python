# sistema-cadastro-python

Sistema em Python executado no terminal que permite cadastrar, listar, buscar e remover usuários, salvando os dados em arquivo para não perder as informações ao fechar o programa.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Busca por nome, email ou ID
- Remoção por ID
- Salvamento em `dados.json`

## Como executar
1. Baixe o projeto (Code → Download ZIP)
2. Abra a pasta no seu computador
3. Execute:

```bash
python main.py
